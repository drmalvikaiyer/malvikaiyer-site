## What this is
A complete redesign of Dr. Malvika Iyer's one-page speaker website (currently live on
WordPress.com; this replaces it). Lilac/plum **jewel** palette, Playfair Display + Lora
serif fonts. Built to deploy free on **Netlify**, with a visual editor for the client.

## How it's built (important)
- **Static site generated by a Python build step.** Source of truth = `content/*.json`.
- `build.py` reads `content/*.json` → writes `index.html`. Pure stdlib (no pip deps).
  - Run locally: `python build.py` (from this folder).
  - On Netlify: build command `python3 build.py`, publish dir `.` (see `netlify.toml`).
- `styles.css` = all styling. `index.html` is GENERATED — never hand-edit it; edit
  `build.py` (markup) or `content/*.json` (content) and rebuild.
- Images live in `assets/`. The build appends `?v=<hash>` to every `/assets/...` URL for
  automatic cache-busting, and to `styles.css`.
- A small inline `<script>` (in build.py, the `FRONT_JS` string) does: stats count-up,
  hero photo cross-fade carousel, "On Stage" sliding marquee, scroll-reveal, and
  instant-jump for the "Connect" links.
